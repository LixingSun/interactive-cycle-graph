# Interactive Cycle Graph

An interactive spiral cycle graph built by [D3.js](https://d3js.org/)

[Demo](https://lixingsun.github.io/interactive-cycle-graph/)

## Getting started
```
// Install Dependencies
npm install

// Start local server
npm run start
```

## Build
```
npm run build
```
